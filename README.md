這是一個類似owncloud 和 網盤的一個小項目

Frontend:Vue2 + ElementUi
Backend:SpringBoot + Mysql + Minio

## 介紹
MinIO基于Amazon S3协议，并提供了与S3兼容的API

## 并且該Demo前後端都采取了分片上傳 （即支持大文件上傳）
